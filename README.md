# Лабораторная работа №1 по дисциплине "Технологии программирования"

## Знакомство с системой Git и CI/CD через GitHub Actions

## Цели работы

- Познакомиться с распределённой системой контроля версий кода Git.
- Изучить принципы непрерывной интеграции (CI) и непрерывного развертывания (CD) в современной разработке ПО.
- Получить практику разработки ООП-программ и написания модульных тестов.
- Освоить работу с системой Git для хранения и управления версиями программного обеспечения.
- Научиться автоматизированному тестированию ПО в Git с использованием GitHub Actions.

## Задание (Вариант 9)

- Формат входного файла: **YAML**
- Определить и вывести на экран студента, у которого по всем дисциплинам 90 баллов.
- Если таких студентов несколько — вывести любого из них.
- Если таких студентов нет — вывести сообщение об их отсутствии.

---
## UML-диаграмма классов

- `DataReader` — абстрактный интерфейс для источников данных.
- `TextDataReader`, `YamlDataReader` — конкретные реализации для разных форматов.
- `StudentsHandler` — обработка и анализ данных студентов.
- `CalcRating` — вычисление рейтинга.
- Тестовые классы проверяют корректность работы каждого компонента.
